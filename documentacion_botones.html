Documentacion de controles

Guia de botones, menus y letreros del proyecto.

Index (Generador de sonido)

Elemento Significado / Accion
Tipo de ondaSelecciona la forma de onda base: seno, cuadrada, triangular o diente de sierra.
Amplitud (volumen)Controla el volumen general del sonido.
Frecuencia (Hz)Define el tono principal en Hertz.
ArmonicosAgrega componentes para enriquecer el timbre.
ReproducirInicia el audio con los parametros actuales.
DetenerApaga el audio de forma inmediata.
Onda / Estado / Frecuencia / ArmonicosLetreros de lectura con el estado actual del audio.
Ir a Sonidos SagradosNavega a la pagina avanzada de instrumentos.

Sonidos Sagrados (Instrumentos)

Elemento Significado / Accion
InstrumentoSelecciona el perfil sonoro: tono puro, flauta, cuenco, gong, dungchen, ruido blanco.
IntensidadControl de volumen general.
Frecuencia baseTono principal sobre el que se construye el instrumento.
ArmonicosNumero de componentes armonicas disponibles.
ReverberacionPorcentaje de mezcla con el efecto de reverb.
Tiempo de decaimientoDuracion del apagado del sonido.
Brillo gongRealza o suaviza las frecuencias altas del gong.
Redondez cuencoSuaviza o endurece el balance armonico del cuenco.
Filtro armonicosEnfasis o atenuacion de armonicos segun el control.
Color del ruidoOscurece o vuelve mas brillante el ruido, segun el instrumento.
ReproducirInicia el sonido con el instrumento seleccionado.
DetenerDetiene el sonido actual.
Instrumento / Estado / Frecuencia / Reverb / DecaimientoLetreros de lectura del estado actual.
Ir a MantrasNavega a la pagina de mantras.
Volver al generador principalRegresa a index.html.

Procedimiento para subir a Github Pages

Pasos para publicar la pagina web desde el repositorio.
  1. Verifica que los cambios esten en la rama main y haz git push.
  2. En Github, entra al repositorio y abre Settings.
  3. En el menu lateral, abre Pages.
  4. En Source, selecciona Deploy from a branch.
  5. En Branch, elige main y carpeta /(root). Guarda.
  6. Espera 1 a 3 minutos y verifica la URL publicada.

URL Normalmente queda como: https://doradoponcedeleon.github.io/generador-de-sonidos-sagrados/
